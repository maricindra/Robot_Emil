Configurando o tailwind

# 1. Digite: npm install -D tailwindcss@3 postcss autoprefixer
Instala 3 pacotes essenciais tailwind(biblioteca que oferece todas as classes )
Postcss ferramenta que apoia o tailwind em processar e transformar o css
Autoprefixer adiciona automaticamente os prefixos de fornecedores necessarios para serem compativeis com todos os navegadores

# 2. Digite: npx tailwind init -p
Comando inicializa o tailwind gerando dois arquivos de configuração:

Arquivo de configuração: Postcss , permite personalizar configurações de css
Arquivo de configuração de tailwind: permite personalizar as configurações do tailwind.
Gera os arquivo: 
# tailwind.config.js e Postcss.config.js

# 6. EXCLUIR O ARQUIVO 'App.css' não usrá porque todos os estilos estaram das classes utilitarias e estilos globais compartilhados do tailwind.
Ou seja, você não escreverá em css tradicial, você aplicará classes utilitarias no html/jsx.

 # ex com app.css
.card-principal {
  padding: 1rem;
  border-radius: 0.5rem;
  background-color: #ffffff;
}

 # ex sem o app.css
<div class="p-4 rounded-lg bg-white">...</div>

# 7. Modificar com regras o fundo da pagina no arquivo index.css, incluindo:
@layer base {
body{ 
@pply bg black text-white relative font-sans h-scren
overflow-x-hidden; }
}

h-screen = Comando ordena que a classe ocupe a posição completa
relative = Para que o corpo tenha um contexto de posicionamento, apoiando ao incluir mais elementos.
overflow-x-hidden = oculta qualquer movimentação horizontal.

# 9. Inclua a classe IMG com as seguintes configurações:

export default function App() {
  return (
    <main>
    
    {/*Imagem de Gradiente */}

    <img className="absolute top-0 right-0 opacity-60 -z-1" src="/public/site_3d_background.png"
    alt=""/>

    </main>
  )
}

Absolute= tira a imagem do fluxo normal, permitindo posicionar em qualquer lugar da pagina
top-0 = posição borda superior de cima
right0 = posição á borda direita
opacity = Torna a imagem semi-transparente
z-1 = empurra a imagem para trás usando sistema index(html)
